 SimpleStudentManager

A small Python console project demonstrating basic object-oriented programming and file handling.  

## Features
- Create, read, and manage student data.
- Uses Python classes and methods.
- Stores data in JSON files.
- Clean `main()` entry point for execution.

## How to Run
1. Clone the repository:
git clone https://github.com/rupom-GH/SimpleStudentManagerForBegginers
.git

2. Navigate to the folder:
cd SimpleStudentManager

3. Run the project:
python main.py

## Requirements
- Python 3.x
